# Task Management App

A full-stack **Task Management Application** built with **MERN stack** (MongoDB, Express, React, Node.js).  
Users can register, login, manage tasks (create, edit, delete, toggle complete), and view task statistics on the dashboard. Profile updates are also supported.

---

## âš¡ Features

- User authentication (register & login) with JWT.
- Create, read, update, delete tasks.
- Toggle task completion status.
- Search tasks.
- Dashboard showing **total, pending, and completed tasks**.
- Update user profile.
- Protected routes for authenticated users.
